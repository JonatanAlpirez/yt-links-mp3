# Reglas:
#   - Una URL por línea
#   - Líneas que empiezan con # o // son comentarios (se ignoran)
#   - Líneas vacías se ignoran
#   - Acepta URL completa, youtu.be/<id>, o solo el ID de 11 caracteres
#   - Texto después de la URL se guarda como descripción opcional
#   - URLs duplicadas se deduplican automáticamente
#
